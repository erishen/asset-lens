#### 1. 配置系统统一
**当前状态**：两套配置系统（config.py + config_manager.py）
**优化方案**：使用 Pydantic BaseSettings 统一配置
**工作量**：2-3 小时
**收益**：可维护性 +20%，配置错误提前发现

```python
# 改进后的配置系统
from pydantic import BaseSettings, validator

class AppConfig(BaseSettings):
    data_mode: str = "sample"
    finnhub_api_key: Optional[str] = None
    
    @validator("data_mode")
    def validate_data_mode(cls, v):
        if v not in ("sample", "real"):
            raise ValueError("data_mode 必须是 'sample' 或 'real'")
        return v
    
    class Config:
        env_file = ".env"
```

from langchain import OpenAI, PromptTemplate

class AIAdvisor:

        self.llm = OpenAI(temperature=0.7)
    
    def generate_advice(self, portfolio_data):
        prompt = PromptTemplate(
            input_variables=["portfolio", "risk_level"],
