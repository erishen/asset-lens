# 🚀 asset-lens：一个用 Python 构建的个人资产操作系统

> 从一个简单的收益记录脚本，到一个完整的资产分析系统

---

## 📌 缘起

最开始，我只是想记录每周的投资收益。

用 Excel 手动统计太麻烦，就写了个简单的 Python 脚本。随着记录的数据越来越多，发现：

- 手动统计效率太低
- 缺乏结构化的分析视角
- 投资行为难以复盘
- 定投策略效果无法验证

于是，我决定把这些零散的脚本重构为一个**长期演进的资产操作系统**。

---

## 🎯 这是什么？

**asset-lens 不是量化交易系统，也不是自动下单平台。**

它是：

- 📊 **个人资产数据管理系统** - 结构化记录你的每一笔投资
- 📈 **收益与风险分析工具** - 年化收益、IRR、最大回撤、波动率
- 🔄 **投资行为复盘工具** - 看清自己的投资决策模式
- 🧪 **策略验证实验平台** - 4 种定投策略模拟
- 🤖 **AI 辅助分析框架** - DeepSeek/OpenAI 智能分析

## 📖 使用指南

### 📊 数据准备

在 `data/real/` 或 `data/sample_data/` 目录下准备 CSV 文件：

**投资记录文件** (`工作表 1-表格 1.csv`)：
```
类型,名称,风险,微信,支付宝,开始日期,初始金额,收益金额,收益率,年化收益,...
货币,余额宝,低,,12580,可赎,,,,,,,,,...
指数基金,沪深300ETF,中,,45000,可赎,2024/1/15,40000,5000,12.5,15.3,...
```

**字段说明：**
- `类型` - 投资类型（货币/基金/股票/理财等）
- `名称` - 产品名称
- `风险` - 风险等级（低/中/高）
- `微信/支付宝/...` - 各平台持有金额
- `开始日期` - 投资开始日期
- `初始金额` - 初始投入金额
- `收益金额` - 已实现收益
- `收益率` - 累计收益率
- `年化收益` - 年化收益率

### 💰 IRR（内部收益率）计算

IRR 是衡量投资真实收益的重要指标，考虑了资金的时间价值：

```bash
# 计算投资组合 IRR
python -m asset_lens calculate
```

**IRR 计算原理：**
- 考虑每笔资金的流入/流出时间
- 计算资金加权收益率
- 比简单收益率更准确反映真实回报

**示例输出：**
```
📊 IRR 分析结果
============================================================
投资组合 IRR:        5.2%
时间加权收益率:      6.8%
资金加权收益率:      5.2%
============================================================
```

### 📝 交易记录格式

交易记录用于记录定投、加仓、减仓等操作：

**格式：** `开始日期-结束日期:操作类型:金额`

**示例：**
```
交易记录
2024/1/1-now:buy:100              # 每周定投 100 元
2024/1/1-2024/6/30:buy:100;2024/7/1-now:buy:150  # 分段定投
2024/1/15-now:buy:100~300         # 智能区间定投（100-300 元）
2024/3/15-now:buy:80-200-400      # 估值模式定投
```

**操作类型：**
- `buy` - 买入
- `sell` - 卖出

### 📈 定投策略使用

支持 4 种定投模式，在 `交易记录` 字段中配置：

#### 1️⃣ 固定金额定投

```
2024/1/1-now:buy:100
```
每周固定投入 100 元，简单省心。

#### 2️⃣ 智能区间定投

```
2024/1/1-now:buy:100~300
```
根据市场估值，投资金额在 100-300 元之间调整：
- 低估时投入 300 元
- 高估时投入 100 元
- 正常估值投入 200 元（平均值）

#### 3️⃣ 浮动金额定投

```
2024/1/1-now:buy:100±50
```
基准金额 100 元，浮动范围 ±50 元：
- 可根据市场情况灵活调整
- 适合有一定经验的投资者

#### 4️⃣ 估值模式定投

```
2024/1/1-now:buy:80-200-400
```
三档估值定投：
- 低估：投入 400 元
- 正常：投入 200 元
- 高估：投入 80 元

**估值模式适合价值投资理念，低位多买高位少买。**

### 🔄 分段定投

可以配置不同时期使用不同策略：

```
2024/1/1-2024/6/30:buy:100;2024/7/1-now:buy:150
```
- 上半年每周定投 100 元
- 下半年每周定投 150 元

### ⚙️ 配置说明

在 `.env` 文件中配置：

```bash
# 数据模式
DATA_MODE=sample          # sample 或 real

# API Keys（用于市场数据和 AI 分析）
FINNHUB_API_KEY=your_key
OPENAI_API_KEY=your_key
OPENAI_BASE_URL=https://api.deepseek.com

# 汇率配置
DEFAULT_USD_RATE=7.1242
DEFAULT_HKD_RATE=0.9157

# 分析配置
MIN_RETURN_THRESHOLD=2.0   # 最低收益率警告阈值
OUTPUT_FORMAT=console,csv  # 输出格式
```
