- [ ] **问题诊断**: `fetch-stock` 命令网络连接失败
  - 错误: `('Connection aborted.', RemoteDisconnected('Remote end closed connection without response'))`
  - 影响: 无法获取实时股票行情
  - 优先级: 高

- [ ] **解决方案**:
  - 添加重试机制 (最大3次，间隔2秒)
  - 支持代理配置 (HTTP/HTTPS代理)
  - 备用数据源降级 (当主数据源失败时)
  - 连接超时设置 (默认30秒)

- [ ] **实施计划**:
  ```python
  # 在 market_stock_fetcher.py 中添加
  def fetch_with_retry(self, stock_code, max_retries=3):
      for attempt in range(max_retries):
          try:
              return self._fetch_stock_data(stock_code)
          except (ConnectionError, TimeoutError) as e:
              if attempt < max_retries - 1:
                  time.sleep(2 ** attempt)  # 指数退避
                  continue
              raise
  ```
